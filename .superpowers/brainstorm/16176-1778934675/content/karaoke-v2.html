字幕渲染 v2 三种描边方案对比

目标：影院级文字清晰度 + 流畅的 Karaoke 高亮动画

方案 A — SVG text 描边 推荐
使用 SVG <text> 的 stroke 属性渲染描边，矢量渲染在任何分辨率下都清晰锐利。填充用 SVG linearGradient 实现逐字扫过。底栏 rgba(255,255,255,0.72) + 模糊。
🎬
方案 B — text-shadow 多层描边
纯 CSS 方案，用 8 方向 text-shadow 构成均匀描边。兼容性最好，但描边在大字号时略粗糙。用 JS 逐字切换 .hl class 实现高亮。
🎬
方案 C — 双层叠加 + -webkit-text-stroke
底层用 -webkit-text-stroke 做描边，上层正常渲染文字。两个层叠加产生清晰的描边效果。实现简单但 -webkit-text-stroke 非 Web 标准。
🎬
附加 — 无底栏纯描边字幕（方案 A 变体）
去掉白色底栏，仅用描边保证在深浅背景上都可读。常见于电影、MV。适合暗色视频背景。
🎬
