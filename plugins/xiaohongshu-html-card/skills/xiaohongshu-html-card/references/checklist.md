# 质量检查清单

生成完卡片后，逐项对照。

---

## 🔴 P0 · 必须通过

### 1. 卡片尺寸正确

- [ ] 每张卡片 canvas 为 1242×1660px
- [ ] 白色卡片为 1151×1580px，居中
- [ ] 卡片圆角 39px，底部边框 5px solid #000

### 2. 字体正确

- [ ] 使用方正兰亭黑系列字体
- [ ] 封面主题词用 Heavy (900)
- [ ] 段落标题用 ExtraBold (800)
- [ ] 正文用 Medium (500)
- [ ] 关键词高亮用 ExtraBold (800) + #d22727
- [ ] 风险提示用 Regular (400) + #2d2727

### 3. letter-spacing 符合规范

- [ ] hero-title: 17.6px
- [ ] cover-subtitle-zh: 8px
- [ ] cover-subtitle-en: 4.5px
- [ ] section-title: 7.68px
- [ ] body-text: 4px
- [ ] disclaimer: 1.4px

### 4. 风险提示

- [ ] 每张卡片底部都有风险提示
- [ ] 风险提示使用 margin-top: auto 固定底部

### 5. 颜色正确

- [ ] 封面背景 #fff1cc
- [ ] 正文背景 #faf3e1
- [ ] 白色卡片 #ffffff
- [ ] 强调色 #d22727
- [ ] 正文黑色 #000000

---

## 🟡 P1 · 排版质量

### 6. 段落间距统一

- [ ] 段落标题与正文间距一致（30px）
- [ ] 正文段落间距一致（15px）
- [ ] 两个段落块间距一致（50px）

### 7. 关键词标记

- [ ] 核心术语已用红色加粗标记
- [ ] 高亮标记不过度（不超过正文的 30%）

### 8. 内容完整性

- [ ] 文案内容完整呈现，无遗漏
- [ ] 列表格式正确（有序/无序）
- [ ] 内容无溢出卡片边界

---

## 🟢 P2 · 视觉打磨

### 9. 布局选择

- [ ] 封面始终是第一张卡片
- [ ] 正文按段落标题合理拆分
- [ ] 每张卡片内容密度适中

### 10. 封面元素

- [ ] 主题词突出（220px）
- [ ] 配图占位区尺寸正确（983×770）
- [ ] 中英文副标题层次清晰

---

## 🧪 最终自检

```
画布与卡片
  □ 1242×1660 canvas
  □ 1151×1580 白色卡片居中
  □ 39px 圆角 + 5px 底边框

字体与排版
  □ 兰亭黑系列，字重正确
  □ letter-spacing 符合规范
  □ 段落间距统一

颜色
  □ 背景色区分封面/正文
  □ 红色强调色正确
  □ 风险提示颜色正确

内容
  □ 所有文案内容完整
  □ 关键词红色标记
  □ 列表格式正确
  □ 每张卡片有风险提示
```
