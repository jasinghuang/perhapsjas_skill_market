# 封面概念图提示词生成

## 这个 Skill 做什么

根据用户输入的文字内容，生成一段可直接用于 AI 绘图工具（Midjourney、DALL-E、Flux 等）的完整提示词。输出的提示词基于预定义的高级概念海报方法论，自动构建视觉隐喻，将词语含义转译为 1:1 比例的图形艺术海报。

## 何时使用

**合适的场景：**
- 小红书封面图需要提示词
- 词语/短句的概念海报提示词生成
- 配合 xiaohongshu-card 生成卡片后，需要封面图提示词

**不合适的场景：**
- 直接生成图片（用图像生成工具）
- 多张图片的系列海报（用 xiaohongshu-card）

## 使用模式

### 模式一：联动 xiaohongshu-card

当用户刚刚使用 xiaohongshu-card 生成了卡片内容后，自动提取变量：

1. **核心文字** — 提取标题或主题词（最能概括内容的 2-6 个字）
2. **文字语言** — 自动检测（中文/英文/中英混合）
3. **补充语境** — 根据文案内容提炼一句简短的语境摘要（20-40 字），概括主题方向、核心观点和传播价值，帮助 AI 绘图工具理解封面要传达的信息
4. **情绪倾向** — 分析文案整体情绪，映射到以下词表：
   冷静、压迫、温柔、危险、纯真、暴烈、克制、疏离、希望、毁灭、浪漫、孤独、秩序、混乱、沉默、对抗、空虚、信念、欲望
5. **禁用元素** — 空置（联动模式不自动填写）

### 模式二：独立使用

用户直接提供一个词/短句，逐个确认变量：
- **核心文字**（必填）
- **文字语言**（自动检测，可覆盖）
- **补充语境**（可选）
- **情绪倾向**（可选）
- **禁用元素**（可选）

## 资源文件

提供三个提示词模板，位于同目录下：

- `prompt-template-withGround.md` — 有承载面版本，强调"承载面 + 角色演绎 + 巨型文字骨架"构图逻辑
- `prompt-template-noGround.md` — 无承载面版本，构图更自由灵活
- `prompt-template-editorial.md` — 编辑设计版本，黑白极简作品集封面基调，文字主导 + 图像隐喻深度融合，支持长标题自动提炼

## 工作流

### Step 1 · 判断使用模式

检测当前会话是否刚用过 xiaohongshu-card 生成了卡片内容：
- 是 → 联动模式，自动提取变量
- 否 → 独立模式，确认变量

### Step 2 · 选择模板

用 AskUserQuestion 让用户选择模板：

```
问题：请选择提示词模板：
选项 A：有承载面版本 — 强调"承载面 + 角色演绎 + 巨型文字骨架"构图逻辑，画面有明确的舞台/地面/平台感
选项 B：无承载面版本 — 构图更自由灵活，没有强制承载面要求
选项 C：编辑设计版本 — 黑白极简作品集封面基调，文字主导 + 图像隐喻深度融合，支持长标题自动提炼
默认推荐：A（有承载面版本）
```

根据用户选择读取对应模板：
- A → `${CLAUDE_PLUGIN_ROOT}/skills/cover-image-prompt/prompt-template-withGround.md`
- B → `${CLAUDE_PLUGIN_ROOT}/skills/cover-image-prompt/prompt-template-noGround.md`
- C → `${CLAUDE_PLUGIN_ROOT}/skills/cover-image-prompt/prompt-template-editorial.md`

### Step 2b · 选择视觉风格（仅模板 C）

当用户选择模板 C（编辑设计版本）时，必须进一步用 AskUserQuestion 询问用户想要的视觉风格。将以下 10 种风格作为选项呈现：

| # | 风格 | 特点 | 适合主题 |
|---|------|------|---------|
| 1 | 弥散风 | 轻盈、柔和、梦幻、空气感强，颜色过渡自然 | 科技想象、品牌氛围感、偏抽象的观点表达 |
| 2 | 马赛克风 | 像被打碎再重组，信息切片、数字噪点、失真感 | 信息过载、互联网观察、认知冲突、数据/媒介/传播、社会议题、数字时代焦虑 |
| 3 | 乐高风格 | 模块化、积木感、结构清晰，天然有"搭建系统"的隐喻 | 教程类、方法论、商业模型、产品拆解、学习路径、成长体系 |
| 4 | 麦肯锡风格 | 理性、克制、清晰、结构化，重点是逻辑感而不是装饰 | 商业分析、行业研究、战略拆解、公司管理、认知框架、方法总结 |
| 5 | 赛博朋克风 | 高密度、霓虹、城市、未来、技术压迫感强（注意避免廉价蓝紫科技感） | Web3、AI、黑客文化、科技社会、虚拟身份、数字资本、未来都市叙事 |
| 6 | 复古风 | 时间感、年代感、情绪感强，容易形成记忆点 | 怀旧、人物故事、品牌故事、内容创作、文化类表达 |
| 7 | 像素风 | 数字感、游戏感、互联网原生感，辨识度高 | Web3、游戏、极客文化、社区、轻内容、有趣教程、年轻化表达 |
| 8 | 酸性设计 / Acid Graphic | 张扬、实验性强、冲突感明显，视觉刺激强 | 前卫观点、亚文化、创意行业、先锋内容、艺术/设计/潮流 |
| 9 | 手绘涂鸦风 | 有温度、有人味、不"AI感"，更亲近 | 个人成长、日常感悟、创作者内容、社群表达、教育类内容 |
| 10 | 极简黑白编辑风 | 克制、现代、排版感强，文字做主角 | 长文封面、概念表达、个人品牌、观点文章、高级感内容栏目 |

用户选择的视觉风格将作为 `{{视觉风格}}` 变量，在 Step 3 中追加到模板 C 的输出末尾。

### Step 3 · 填充变量

将提取/确认的变量填入模板中的占位符。

**模板 A / B 的变量：**

| 占位符 | 联动模式 | 独立模式 |
|--------|---------|---------|
| `{{核心文字}}` | 提取标题/主题词 | 用户输入 |
| `{{文字语言}}` | 自动检测 | 自动检测 |
| `{{补充语境}}` | 从文案中提炼的简短语境摘要 | 用户补充（可选） |
| `{{情绪倾向}}` | 分析文案情绪 | 用户指定（可选） |
| `{{禁用元素}}` | 空置 | 用户指定（可选） |

**模板 C 的变量：**

| 占位符 | 联动模式 | 独立模式 |
|--------|---------|---------|
| `{{主题词}}` | 提取标题/主题词 | 用户输入 |
| `{{副标题}}` | 从文案中提取副标题 | 用户输入（可留空） |
| `{{画幅比例}}` | 默认 1:1 | 用户指定（默认 1:1） |
| `{{文字语言}}` | 自动检测 | 自动检测 |
| `{{用途}}` | 根据上下文判断（默认"海报"） | 用户指定 |
| `{{补充语境}}` | 从文案中提炼的简短语境摘要 | 用户补充（可留空） |
| `{{情绪倾向}}` | 分析文案情绪 | 用户指定（可留空） |
| `{{禁用元素}}` | 空置 | 用户指定（可留空） |
| `{{视觉风格}}` | Step 2b 用户选择 | 用户选择 |

### Step 4 · 输出提示词

将填充完成后的完整提示词输出给用户，可直接复制到 AI 绘图工具使用。
