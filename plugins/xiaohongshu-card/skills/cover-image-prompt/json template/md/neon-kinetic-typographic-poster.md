# 霓虹动感排版海报

## 风格概述

戏剧性的户外编辑海报风格，结合低角度生活摄影、超大变形霓虹排版、胶片颗粒和高能量青年文化活动设计。

## 环境变量

| 变量名 | 中文说明 |
|--------|---------|
| SUBJECT | 主要人物或物体 |
| SUBJECT_ACTION | 主体的动态动作 |
| PRODUCT_OR_PROP | 主体手持、佩戴、使用或展示的物品 |
| LOCATION | 户外环境或场景背景 |
| BACKGROUND_ELEMENTS | 天空、山丘、建筑、人群、舞台灯光等次要场景元素 |
| MAIN_TEXT | 大号重复出现的标题词或短语 |
| SECONDARY_TEXT | 小型重复文字行 |
| ACCENT_SYMBOL | 小型分隔图标（菱形、圆点、斜杠、星号等） |
| PRIMARY_GRAPHIC_COLOR | 电光霓虹黄 |
| PHOTO_COLOR_TONE | 温暖阳光自然色调配蓝天对比 |
| WARDROBE_STYLE | 街头服饰、户外时尚、运动装、音乐节穿搭或编辑风格 |
| CAMERA_ANGLE | 极端低角度广角透视 |
| ASPECT_RATIO | 9:16 竖版海报 或 16:9 横版横幅 |

## 构图

| 字段 | 中文说明 |
|------|---------|
| layout_type | 带排版边框系统的摄影海报构图 |
| subject_position | 主体在中下方或略偏中心，从下方仰拍使身体向上延伸 |
| typography_position | 超大标题排版围绕边缘并在主体后方，被画布裁切 |
| text_scale | 标题字母应极大，经常大到无法完全放入画面 |
| text_behavior | 字母可以拉伸、剪切、扭曲、弯曲、压缩或环绕构图 |
| depth_layering | 摄影主体出现在部分排版前方，其他文字在主体后方或周围 |
| negative_space | 开阔天空或干净背景区域帮助霓虹排版保持可读性 |
| cropping | 鼓励对字母、肢体、道具和背景边缘进行大胆裁切 |
| motion_feel | 构图应感觉不稳定、动感、响亮、充满运动感 |

## 排版

| 字段 | 中文说明 |
|------|---------|
| headline_font_style | 超粗压缩展示字体，定制变形字型，锐利切口，斜体倾斜，拉伸垂直形态 |
| secondary_font_style | 细体大写无衬线体，几何感，稀疏间距，极简，水平重复 |
| headline_treatment | 霓虹实色填充，无描边，与天空或景观形成极高对比 |
| secondary_text_treatment | 小型重复大写文字行，以点缀符号分隔 |
| letterform_character | 棱角分明、运动感、扭曲感、攻击性、海报级尺度 |
| readability_rule | 标题可被部分裁切，但必须保持视觉上可识别为图形语言 |

## 摄影指导

| 字段 | 中文说明 |
|------|---------|
| camera | 广角镜头，贴近地面，向上倾斜 |
| lighting | 硬朗自然阳光，暖色高光，深色阴影，户外曝光 |
| texture | 可见胶片颗粒，轻微印刷噪点，轻微灰尘，模拟编辑质感 |
| lens_effect | 广角镜头的微妙畸变，夸张靠近镜头的肢体或道具 |
| mood | 原始、自信、年轻、充满活力、阳光充沛 |

## 图形元素

| 字段 | 中文说明 |
|------|---------|
| primary_type | 巨大的霓虹标题文字 |
| secondary_type_band | 小型重复大写短语水平排列 |
| separator_symbols | 重复词之间的小型霓虹点缀符号 |
| edge_usage | 字母和符号应延伸超出画布边缘 |
| overlay_rule | 图形可部分覆盖背景，但不应遮挡关键面部、动作或产品（除非刻意编辑效果） |
