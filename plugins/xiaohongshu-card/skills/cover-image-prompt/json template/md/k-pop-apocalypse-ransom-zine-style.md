# K-pop 末日拼贴杂志风格

## 风格概述

极繁主义 K-pop 时尚杂志拼贴风格，以中央人像剪裁为核心，搭配揉皱黑白纸张纹理、倾斜勒索信排版、喧闹贴纸色块、饱和荧光/蓝/红强调色和粗体底部刊头色带。

## 风格类别

- K-pop 时尚杂志海报
- Y2K 流行朋克拼贴
- 末日风编辑封面
- 勒索信排版杂志版面
- DIY 街头时尚传单

## 核心视觉识别

### 构图

主体是大型中央剪裁，带有厚撕裂纸白边。超大排版围绕并在主体后方，经常旋转、裁切或部分隐藏。小型编辑说明填充侧边距。饱和的底部刊头色带锚定画面。

### 主体位置

主体放在中下或中右。紧裁头部和上身。让头发、肩膀和剪裁纸边缘突破周围文字块。

### 相机/透视

近距肖像构图，略低或正面相机，时尚编辑凝视，看向画外，随意不完美头发动态和闪光灯皮肤质感。

### 排版

使用厚重变形无衬线体、贴纸风格方块字母、倾斜标签、混合字号和模拟勒索信叠层。主要文字应有粘贴感、倾斜感和喧闹感，而非工整排版。

### 图形元素

- 撕裂白色肖像背板
- 斜向荧光绿横幅
- 蓝色贴纸标签
- 红色星星盖在一只眼旁或面部附近
- 黑白波普风格文字块
- 侧边小型编辑说明
- 底部刊头色带配缩略图内嵌
- 复印颗粒和纸张褶皱

### 质感与完成效果

完成效果应像物理组装后扫描：揉皱纸张、复印噪点、半色调斑点、不平整边缘、轻微印刷偏移、粗糙阴影和不完美图层对齐。

### 光线

在主体上使用硬朗杂志闪光灯，嘴唇、脸颊和配饰上有光泽高光。背景保持更平、更暗、更有触感。

### 情绪

混乱、俏皮、叛逆、前卫时尚、末日感、流行朋克、讽刺、年轻、编辑感。
