# 毛绒城市节庆手机海报

## 风格概述

明亮活泼的手机活动海报风格，结合真实城市地标、柔软毛绒吉祥物角色、圆角应用卡片 UI 边框、粗体白色节庆排版、日期/地点文字和友好的旅游宣传氛围。

## 环境变量

| 变量名 | 中文说明 |
|--------|---------|
| EVENT_NAME | 活动名称（节日、市集、公共活动或季节性推广） |
| CITY_OR_DISTRICT | 城市/地点（城市、街区、公园、广场等） |
| PRIMARY_MASCOT | 大型毛绒或蓬松吉祥物角色 |
| SECONDARY_MASCOT | 小型伴侣吉祥物、毛绒动物或玩具宠物 |
| MASCOT_ACTION | 吉祥物动作（跑、挥手、漂浮、引导、跳舞等） |
| LANDMARKS | 可辨识的城市建筑、塔楼、广场等地标 |
| SKY_AND_WEATHER | 明亮蓝天、柔和云层、晴天 |
| GROUND_PLANE | 人行横道、广场、公园小径、水滨步道等地面元素 |
| DATE_RANGE | 大型活动日期 |
| LOCATION_TEXT | 小型地点标注 |
| CTA_TEXT | 简短手机端行动号召文案 |
| TOP_MICRO_TEXT | 顶部小标签或节庆系列文字 |
| APP_UI_ELEMENTS | 移动端 UI 元素（返回箭头、三点菜单、底部弹窗、圆角卡片等） |
| COLOR_ACCENTS | 吉祥物颜色、清澈天蓝、白色排版、深灰页脚、柔和城市中性色 |
| ASPECT_RATIO | 9:16 竖版手机海报 或 16:9 横版活动横幅 |

## 构图

| 字段 | 中文说明 |
|------|---------|
| layout_type | 带圆角的移动端应用活动卡片海报 |
| main_card | 占据画面大部分的大型圆角顶部视觉卡片 |
| headline_position | 居中大标题位于天空区域上方 |
| headline_scale | 足够大，手机屏幕上能瞬间看清 |
| mascot_position | 吉祥物在中下方前景跑动或漂浮，覆盖城市街道或广场 |
| landmark_position | 地标在吉祥物后方，排列成欢快的城市全景 |
| date_position | 大号日期范围在右下视觉区域或底部卡片边缘 |
| location_position | 小型地点文字在左下角 |
| bottom_sheet | 白色圆角手机底部面板，带拖拽手柄和行动号召文案 |
| depth_layering | 天空在标题后方 → 地标在中间 → 吉祥物在前景 → UI 文字在最上层 |
