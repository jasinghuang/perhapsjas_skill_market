# 组件协议、raw 逃生舱与填满策略

借鉴 beautiful-article 的"语义组件 + raw 逃生舱 + 主题 token"理念，make-html-card 用更轻的
等价物：**固定布局模板（占位符填空）+ 可选 raw 逃生舱 + 主题 CSS 变量 + 填满策略**。

## 默认：按布局模板占位符填空（不裸写样式）

AI 默认只做"占位符替换"——读对应 `assets/layouts/<type>/default.html`，把内容填进
`{{TOPIC}}` / `{{SECTION_TITLE_N}}` / `{{BODY_PARAGRAPH_N}}` 等占位符，**不改动模板的 CSS**
（字号 / 字重 / 颜色 / 间距已写成主题变量，你只填内容）。结构与排版由模板保证，
AI 不需要、也不应该去想怎么实现排版。

## 填满策略（实战核心）

内容框 1335px（padding 70 到风险提示 1405），目标填充 **~92%**（约 1228px，满但不挤风险提示）。

**铁律：行距（line-height 1.42）固定不动（易读性），填满靠段落间距（margin）+ 视觉元素。**

1. **行距 line-height 1.42 固定**（design-tokens 规范）—— **不得为填满调行距**（破坏易读性）。
2. **段落间距（margin）是填满杠杆** —— 块间 50 / 段间 15 / 标题→正文 30（design-tokens 规范）。
   精调填充率（±几 px）改 margin/padding，不改内容（内容块 ~140px 级，加减跳 ~10%，难精确卡中间值）。
3. **视觉元素填充** —— 内容偏少时，加**重点框**（Aside 风格，红边）/ **引用框**（暖黄底，核心洞察）/ 数字卡片 / 网格，用视觉补足（不只靠正文）。
4. **不要撑字号 / 行距填满** —— 那是破坏易读性的粗暴做法。

> 填满度精调：内容决定"有没有"（信息），间距决定"满不满"（填充率）。要 88/90/92%，调 margin；
> 要大幅变化（±10%），才动内容块。

## raw 逃生舱：本卡需要特殊视觉时

当某张卡需要一个模板占位符装不下的视觉——数据图表、流程图、对比图、特殊装饰、自定义布局块——
用 raw 逃生舱。位置在 content 布局模板里被注释的 `raw-block` 区（正文 section 之后、spacer 之前）：

1. 取消注释 `<div class="raw-block"> … </div>`
2. 在内部填入任意 HTML / 内联 CSS / SVG（对应 `{{RAW_BLOCK}}` 占位符）
3. 不需要时整段保持注释，或直接删掉

### raw 的硬约束（主题 token 兜底）

raw 给自由，但必须守住主题统一：

1. **样式只用主题变量**：颜色用 `var(--xhs-accent)` / `var(--xhs-text-primary)` / `var(--xhs-card-bg)`
   等；字号用 `var(--xhs-body-size)` 等。**不得写野生色值 / 字号**——否则切换主题时 raw 不会跟着变。
2. **不得越界**：raw 内容不得超出白色卡片 1151×1580，**不得遮挡 `.disclaimer`**（风险提示 absolute
   钉底，合规红线，见 `repair-policy.md`）。
3. **不得改规格**：字体族仍是方正兰亭黑系列；卡片尺寸 / 圆角 / 边框不动；风险提示文案不动。
4. **服务内容**：raw 必须服务本卡的论点 / 数据 / 图解，不是装饰。一张卡最多一个 raw 块，
   且不得挤掉正文（**正文仍是主体**，raw 是增强）。

## 四种"视觉需求"的路由

| 需求 | 用什么 |
|---|---|
| 普通正文 / 列表 / 关键词高亮 | 模板占位符（默认） |
| 封面配图 | cover 模板的 `{{COVER_IMAGE}}` |
| 重点框 / 引用框（填满用） | html 元素（key-point 红边 / quote-box 暖黄底） |
| 图表 / 流程图 / 对比图 / 特殊装饰 | raw 逃生舱（`{{RAW_BLOCK}}`，受上方硬约束） |

## 自检（Card Reviewer 查）

- 填充度 ~92%（满但不挤风险提示）？
- 行距 line-height 1.42（没调）？段落间距统一？
- 视觉元素（重点框/引用）服务内容，不是堆砌？
- raw 是否用了 `var(--xhs-*)`，没野生色值 / 字号？
- raw 是否越界 / 遮挡风险提示？
- 本卡正文仍是主体？
