# 质量检测标准

> 适用于所有分类的全局质量检测清单。各分类的 `standards.md` 可覆盖或补充本文件。

## 一、合规性检查

- [ ] 未包含具体股票/基金推荐代码
- [ ] 未承诺或暗示具体收益
- [ ] 涉及投资相关内容时，包含风险提示
- [ ] 未使用绝对化用语（"一定赚"、"稳赚不赔"等）
- [ ] 内容不构成投资建议的表述

## 二、标题质量

- [ ] 封面标题足够吸引眼球，文字极简
- [ ] 文字标题不超过 20 字
- [ ] 文字标题包含重要关键词（SEO 优化）
- [ ] 标题不剧透核心内容

## 三、正文描述（Description）

- [ ] 30-50 字左右
- [ ] 包含重要关键词（SEO 优化）
- [ ] 未剧透最核心的方法/结局
- [ ] 包含 3-4 个相关 Tag（#标签格式）

## 四、正文内容

- [ ] 开头有"黄金3秒"钩子（引起好奇心或共鸣）
- [ ] 结尾有明确的 CTA（引导互动：点赞/收藏/评论/关注）
- [ ] 段落简短，适合手机阅读
- [ ] 逻辑清晰，有完整的信息链
- [ ] 无错别字、语病
- [ ] 数字、数据准确无误（如有引用需可追溯）

## 五、人设一致性

- [ ] 语气风格符合 `personality.md` 的设定
- [ ] 自称方式一致
- [ ] 专业程度与目标受众匹配

## 六、爆款逻辑

- [ ] 有情绪起伏（非平铺直叙）
- [ ] 有痛点/痒点/爽点中的至少一个
- [ ] 提供了可感知的价值（信息增量/情绪价值/实用工具）
- [ ] 内容有"转发动力"（值得分享给他人）
