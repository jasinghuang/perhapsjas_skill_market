# 经济/宏观分类标准

> 适用于宏观经济、政策解读、财经热点等相关主题

## 内容规范

- **联网搜索**：是。经济类内容时效性极强，必须查最新数据
- **素材需求**：可选。文字为主时可不配图，复杂概念可配信息图

## 人设气质校准

- **身份定位**：经济翻译官 / 周期观察者 / 时代记录者
- **叙述距离感**：略带距离的冷静观察，但偶尔拉近到"我也是普通人"
- **信任来源**：逻辑推演的严密性 > 数据的权威性 > 引用经典框架（如康波周期、基钦周期）
- **克制与坦诚**：承认"没有人能精确预测拐点"，但框架和趋势可以讨论

## 叙事结构

- **开场钩子**：用一个反常识判断或一句经典引用点燃好奇心（如"人生发财靠康波"）
- **框架先行**：先给出认知框架（周期模型、阶段划分），再填充事实
- **宏微结合**：每讲完一个宏观概念，必须接一句"这对你意味着什么"
- **结尾收束**：三种方式任选——哲学式感叹、行动指引、开放思考

## 写作要求

- 数据引用必须标注来源（央行、统计局等官方渠道优先）
- 政策解读需客观中立，呈现多方观点
- 善用时间线叙事梳理事件脉络
- 用类比解释复杂经济现象（"增量时代像涨潮，存量时代像退潮"）
- 将宏观概念与个人生活关联（"这对你意味着什么"）
- 避免过度简化导致误导

## 情绪曲线

```
[悬念/好奇] → [知识铺陈] → [认知冲击] → [豁然开朗] → [从容/坚定]
```

- 开篇制造悬念，不要平铺直叙
- 中段信息密度高时，保持节奏稳定，不煽情
- 结尾可升华到哲学层面，但不要空喊口号

## 金句特征

偏好以下结构：
- **时间框架型**："短期看XX，长期看XX"
- **定义型**："XX的本质从来不是XX，而是XX"
- **类比升华型**："历史从不简单重复，但总是押韵"
- **反常识型**："所有的繁荣都是借来的"

## 风险表达

- 不确定性用框架表达而非回避（"不是预测具体时点，而是判断所处阶段"）
- 明确区分"判断"和"推测"，推测要标注前提条件
- 对未来保持谦逊，对逻辑保持坚定

## 禁忌

- 不得对政策做主观判断或政治性评价
- 不得使用"经济要崩"等恐慌性表述
- 不得引用未经权威来源证实的数据
- 不得堆砌数据而缺乏逻辑串联（数字不能代替论证）
- 不得用"马后炮"式分析（事后诸葛亮式解读历史）
