# 经济/宏观分类标准

> 适用于宏观经济、政策解读、周期分析、财经热点、社会阶层等相关主题
> 框架来源：finance-article-deconstruction-card.md
> 基于 5 篇示例文章的跨文章模式分析（SOL x2、Nico x1、大神王 x1、TrungTPhan x1）

---

## 基础信息

- **主题领域**：宏观趋势、政策解读、经济周期、社会阶层、财经热点、认知框架
- **文章类型偏好**：观点评论型（40%）> 框架拆解型 > 攻略教程型 > 案例复盘型
- **联网搜索**：是。经济类内容时效性极强，须查最新数据和政策
- **素材需求**：可选。文字为主时可配信息图，短文可不配

---

## 核心观点与论证

- **核心观点表达方式**（按实际出现频次排序）：
  1. 反共识先声夺人（80%）：几乎每篇都用——"年轻人不要负债""四大经济周期同时转向""兴趣广泛是一种超能力""穷人没教育寒门无贵子"
  2. 框架重新定义（60%）：用周期理论/阶段划分/结构分析重构读者的认知地图
  3. 故事引出洞察（60%）：三亚的朋友、质检员、五金老板
  4. 数据推翻常识（60%）：辈出率数据、家庭教育支出、票房数据
- **核心观点表达时机**：
  - 观点评论型：开头直接抛出结论（"年轻人不要负债"），中段论证，结尾升华
  - 框架拆解型：先制造认知缺口（"你以为的行情波动背后是结构变化"），再用框架填补
  - 攻略教程型：先建立共鸣，再翻转认知，最后给路径
- **用于论证的工具**：
  - 最强组合：历史类比 + 逻辑推演 + 具体数字（3/5长文都用这个三角组合）
  - 逻辑推演链条：不是数据堆砌，而是"让读者自己走一遍推导过程"——A→B→C→D，而D正是你正在经历的事情
  - 历史类比：1929大萧条、2008金融危机、文艺复兴、互联网/淘宝/抖音的窗口期。标注类比局限
  - 统计数据：辈出率、教育支出比例、哈佛/清华入学数据。来源标注官方渠道
  - 宏观框架：增量vs存量博弈、四大经济周期（基钦/朱格拉/库兹涅茨/康德拉季耶夫）

## 叙事结构脉络

四种类型对应四种结构：

### 观点评论型（最主流）
```
钩子（名人名言/个人故事）→ 现象描述 → 反常识判断 → 宏观论证（至少3层递进：现象→结构→本质）→ 哲学升华 → 行动指引
```

### 框架拆解型
```
钩子（非共识判断）→ 框架总览 → 逐层拆解（短→中→长→叠加）→ 叠加分析 → 结论（历史不重复但押韵）
```

### 攻略教程型
```
痛点共鸣 → 认知翻转 → 理论基础 → 历史类比 → 实操路径 → 具体步骤
```

### 案例复盘型（短文专用）
```
悬念（"最重要的午餐"）→ 事件叙述 → 关键数据 → 起源故事（时间线）→ 一句话结论
```

- **共同底层节奏**：所有长文都遵循"制造认知缺口 → 用框架/数据填补 → 给出情绪出口 → 以行动或哲学升华收束"
- **结尾收束**：不要空喊口号。三种收束任选——哲学金句、行动指引、紧迫感收尾。不能说"你怎么看"或"欢迎讨论"——这是逃避，作者必须给出自己的判断

## 人设气质档案

四位作者展现不同人设，但共享一个底色——**"过来人"而非"学术权威"**，知识来源于实践和思考。

### 共性特征
- 平等的对话者，不是高高在上的导师
- 都有意识地展示"我也不全知道"——坦诚边界比全知全能更能建立信任
- 都有"只有亲历者才有的细节"——三亚朋友故事、"我的灵感和产品系统"、群友咨询白酒股票

### 四种人设方向
| 方向 | 代表 | 核心气质 |
|------|------|---------|
| 系统批判者/哲学家 | SOL（01） | 从经济现象推演到哲学本质（"债务是时间的异化"），冷静但内容激进 |
| 周期观察者/分析师 | SOL（02） | 框架感最强，信息密度最高，个人情绪最少 |
| 实操教练 | Nico | 最具行动感，大量使用第二人称"你"，给出具体步骤和工具 |
| 真诚的焦虑者 | 大神王 | 最具个人叙事色彩，故事驱动，情绪起伏最大，坦诚程度最高 |
| 极简叙事者 | TrungTPhan | 零评论、零情绪，完全靠数据和事实说话，一句话结论 |

- **与读者的关系体感**：略带距离的冷静观察者，偶尔拉近到"我也是普通人"
- **克制与坦诚**：承认"没有人能精确预测拐点"，但框架和趋势可以讨论

## 阅读氛围体感

- **整体情绪关键词**：清醒、宏大、有脉络、不焦虑
- **情绪曲线**（通用四拍节奏）：
  ```
  [不安/悬念] → [加深/认知冲击] → [翻转/重建] → [释放/行动感]
  触发因素：开场钩子 / 论证深化 / 框架或数据填补 / 行动方向或哲学升华
  ```
- **情绪触发器统计**：
  - 时代对比（过去vs现在）：使用率80%
  - 认知反转（"你以为…其实是…"）：使用率60%
  - 具体人物故事：使用率60%
  - 数字冲击：使用率60%
- **说服力来源**：逻辑严密感 > 数据权威感 > 人格信任感
- **认知节奏**：中等密度，框架展开时信息密集，宏微转换处给消化空间。不煽情
- **信任建立方式**：独立思考态度 > 诚实承认预测局限 > 引用经典框架的准确度
- **读者的"安全感"**：认知升级——"我终于理解为什么了"，而非"我该怎么做"
- **焦虑管理铁律**：可以让读者不安，但文章结束前必须给出至少一个"你可以做什么"的方向。希望必须建立在具体的逻辑或事实之上，不能空洞安慰

## 案例/数据生命点

- **核心案例的画面感**：历史事件的戏剧性瞬间、身边人的命运对比（"三亚的朋友""质检员""五金老板"）
- **数据讲述方式**：冰冷的数字要变成有温度的故事——不是"GDP增长X%"，而是"这意味着每个打工人的工资……"。使用数据时必须同时给出参照物
- **最具冲击力的对比**：增量时代 vs 存量时代的切身感受对比、辈出率从1.15下滑到0.66
- **案例与结论的连接方式**：归纳总结为主（多个历史案例指向同一规律），逻辑推演为辅（A→B→C→D链条）

## 市场语境适配

- **文章与市场情绪的关系**：优先选择「抽离观察」（冷静分析不理会情绪），次选「逆风提醒」
- **时效性处理**：周期类分析长期有效，政策类需标注时点。明确告诉读者"本文讨论的是框架，不是短期预测"
- **紧迫感的正确用法**：不用"再不XX就来不及了"，而用"窗口在以你感觉不到的速度收窄"——后者更有说服力因为它承认了感知偏差

---

## 写作技术痕迹

- **标志性句式**：
  - "过去人们负债是为了享受通胀的红利，现在人们负债是为了被通胀惩罚"（对比型）
  - "所有的繁荣都是借来的"（反常识型）
  - "短期来看，市场是情绪。长期来看，市场是周期"（时间框架型）
  - "债务不再是一种选择，而是一种宿命"（否定+重定义型）
  - "不要在那个愣住的感觉里待太久。那个感觉，待久了，就成了借口"（行动号召型）
- **过渡手法**：从数据跳到观点时，用一句"换句话说"或类比桥接；从观点跳到行动时，用"这对你意味着什么"连接
- **金句分布规律**：开篇1句定调 + 每个论证转折处1句 + 结尾1句升华
- **金句特征**：
  - 来源类型：市场规律提炼、历史类比化用、反直觉洞察
  - 结构特点：最有力量的金句几乎都用**"否定+重定义"结构**（"A不是X，而是Y"）和对比结构
  - 密度：每1000字1-2句，不要超过3句——金句过密等于没有金句
  - 短文（案例复盘型）刻意不用金句，纯靠事实——这是一种克制

## 风险表达方式

- **不使用模板化免责语言**（"投资有风险"类）——所有高质文章都不用
- **风险表达融入论证本身**，而非单独成段：通过论证"债务是陷阱"来表达风险；通过"窗口在收窄"来表达紧迫感
- **最高级的风险表达是"承认不知道"**："我没有资格给你一套完整的解决方案"——这反而建立了最大的信任
- **时效性风险通过"窗口"隐喻表达**："LLM才三岁……但不会一直三岁"
- **明确区分"判断"和"推测"**：推测要标注前提条件。对未来保持谦逊，对逻辑保持坚定

## 给AI的创作方向

- **语气指引**：一个看透了的过来人在和你平等对话——不是教导，不是卖弄，是分享洞察。写宏观分析时冷静克制，写社会议题时带情绪但不失理性，写个人发展时激励但不鸡汤
- **结构指引**：
  - 所有长文必须遵循：钩子 → 不安 → 论证/框架 → 翻转/希望 → 行动/升华 的五段节奏
  - 观点评论型的论证链至少3层递进（现象→结构→本质），每层都要有具体证据
  - 框架拆解型必须先给出全景框架，再逐层拆解，最后回到全景叠加分析
  - 段落长度：不能连续3段以上纯理论无案例/故事——认知节奏必须有"理论-例证"的交替
- **人设指引**：
  - 身份不是分析师，不是学者，是"深度思考的实践者"
  - 每篇至少有一处"我不全知道"或"我没有资格"的表达
  - 每篇至少有一个"只有亲历者才有的细节"——具体的、不可复制的观察
  - 系列文章间可互相引用，建立连续性和深度感
- **专业度校准**：
  - 术语密度：每千字不超过3个专业术语，每个术语后面紧跟人话解释
  - 数据使用：数据不是用来堆砌的，是用来"让读者看见规模感"的
  - 引入框架时先说"为什么需要这个框架"，再说框架内容
- **金句设计规则**：
  - 优先使用"否定+重定义"结构（"A不是X，而是Y"）和对比结构
  - 金句必须从论证中自然生长出来，不能凭空插入
- **需要避免的雷区**：
  - 使用模板化免责语言（"投资有风险"类毫无价值的表述）
  - 用数据吓人但不给出路——每制造一次焦虑，必须在3段内给出认知框架或行动方向
  - 用"众所周知"开头——如果众所周知就不需要写
  - 连续3段以上纯理论无案例/故事
  - 在结尾说"你怎么看"或"欢迎讨论"
  - 空洞的正能量（"一切都会好的"）——希望必须建立在具体逻辑或事实之上
  - 堆砌数据而缺乏逻辑串联（数字不能代替论证）
  - 对政策做主观判断或政治性评价
  - 引用未经权威来源证实的数据
