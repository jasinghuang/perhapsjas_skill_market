# 投资分类标准

> 适用于股票、基金、投资策略、资产配置、市场分析等相关主题
> 框架来源：finance-article-deconstruction-card.md
> 基于 10 篇示例文章的跨文章模式分析（Nico x2、SOL x4、Luna x2、innomad x1、口播 x1）

---

## 基础信息

- **主题领域**：投资策略、基金/股票科普、资产配置、全球投资、实操指南
- **文章类型**：攻略教程型（50%）、框架拆解型（40%）、权威解读型（10%）。权威解读型不靠个人实战经历，靠精确引用大师材料（股东信/演讲/著作）建立信任
- **联网搜索**：是。需要最新市场数据、基金费率、开户政策等
- **素材需求**：是。数据图表、对比表格、流程图增强说服力

## 核心观点与论证

- **核心观点表达方式**（按实际出现频次排序）：
  1. 框架重新定义（44%）：把复杂投资决策封装成可执行框架——"三要三不要""四笔钱""72法则""三个买入方法"
  2. 反共识先声夺人（22%）："亏损2000万后的经验""美股三权分立是最硬的底裤"
  3. 故事引出洞察（17%）：2008金融危机复盘、401k法案的故事
  4. 数据推翻常识（17%）：1900-2012各类资产收益对比、策略A vs 策略B
- **核心观点表达时机**：
  - 教程型：痛点对比后直接引出（"以前觉得难，其实3步搞定"）
  - 框架型：开头必须有钩子——名言/亏损经历/反常识结论，再展开论证
- **用于论证的工具**（按威力排序）：
  - 最强组合拳：历史数据 + 权威引用 + 反事实对比。三篇被引用最多的文章都同时使用了这三种
  - 历史数据回溯：标普500历史回撤、各类资产收益对比、纳斯达克20年数据。必须注明来源
  - 权威引用：巴菲特/格雷厄姆/段永平/DALBAR研究/Brinson研究。引用后跟一句大白话翻译
  - 反事实对比：策略A vs 策略B（早投vs晚投）、持有vs空仓的收益差——让读者"算出来"差距
  - 个人实操案例：港卡开户实测、个人配比方案、持仓比例展示

## 叙事结构脉络

两种结构模板，按文章类型选择：

### 攻略教程型结构（Nico/Luna/innomad）
```
开场（1-3句话直接进入主题）→ 分步教程/配置建议 → 个人方案展示 → 风险提示融入操作中 → 收束引导
```
- 不制造焦虑，制造"这件事没你想的那么难"的安全感
- 文末统一引导关注

### 框架拆解型结构（SOL/Luna理念篇）
```
钩子（名言/亏损经历/反常识结论）→ 框架建立 → 历史/数据填充 → 执行方案 → 哲学性升华结尾
```
- 认知节奏：密集→松→密集→升华，像呼吸一样
- 结尾上升到"投资哲学"或"生活方式"的高度

- **开场钩子**：个人亏损经历（"亏了XX万那天晚上翻来覆去"）、颠覆常识的判断、或直接给结论
- **结尾收束**：行动指引为主（"今天就可以做XX"），让读者有路径感。框架型可用哲学金句

### 权威解读型结构（口播文案/大师解读）
```
矛盾钩子（大师的某个"反常"行为）→ 层层追问（为什么→那为什么→这个例外呢）→ 大师原话+翻译 → 坦诚大师也会犯错 → 原则收束
```
- 开场制造认知矛盾——大师做了看似矛盾的事（"不碰科技股却重仓苹果"）
- 用"追问"推进而非"推演"推进：每个章节回答一个读者的自然追问
- 每段权威引用后必须跟一句"换句话说"大白话翻译
- 必须包含"大师也会犯错"段落，避免造神
- 结尾回到大师的核心原则/框架，形成闭环

## 人设气质档案

四位作者展现四种不同人设，但共享同一个底色——**"我也是普通人，只是多走了几步"**，没有一人以专业投资机构人士自居。

### 共性特征
- 都展示个人持仓/操作经历（Nico的5张港卡配置、SOL的2000万亏损、Luna的80/20配比、innomad的核心-卫星策略）
- "过来人"姿态，非"老师教学生"，更像"老司机带新手"
- 不预测点位、不推荐个股、不承诺收益，坦诚自己的亏损经历

### 五种人设方向（按需选择）
| 方向 | 代表 | 适用文章类型 | 语言风格 |
|------|------|-------------|---------|
| 操作执行层 | Nico | 攻略教程型 | 干净利落、步骤化、少废话 |
| 市场认知层 | SOL | 框架拆解型 | 口语化、有力量感、善用比喻 |
| 心理建设层 | Luna | 理念+实操结合 | 温暖、耐心、善用生活类比 |
| 知识体系层 | innomad | 系统教程型 | 冷静、结构化、学术味略重 |
| 权威翻译层 | 口播文案 | 权威解读型 | 口语化、追问式、翻译大师语言 |

### 权威解读型人设的补充说明
- **信任来源不同**：不靠"我也亏过"的实战痕迹，靠精确引用大师材料（股东信年份、原文、具体数据）
- **身份定位**：大师的"翻译者"——"我来帮你搞懂巴菲特/芒格/达利欧到底在想什么"
- **与读者的关系**：知识导游，非过来人。不展示个人持仓，但展示对大师材料的深度熟悉
- **必须有的克制**：包含"大师也会犯错"段落（如巴菲特的IBM），避免造神

- **专业痕迹**：能准确引用PE百分位、最大回撤、年化收益，但不堆砌。引用后跟"换句话说"
- **实战痕迹**：能写出"亏了XX万那天晚上翻来覆去"——只有真金白银体验才有的身体记忆
- **克制与坦诚**：明确说"我无法预测底部在哪"，承认无知反而是可信度来源

## 阅读氛围体感

- **整体情绪关键词**：清醒、务实、有路径、不焦虑
- **情绪曲线**（三种）：
  ```
  曲线1："安全降维"（教程型）
  [焦虑："我不会/不敢"] → [快速消除："其实很简单"] → [步骤化给予掌控感] → [温和收束]

  曲线2："认知冲击→重建"（框架型）
  [认知挑战："你以为的X其实是Y"] → [数据/历史证明] → [新框架建立] → [读者获得升级感]

  曲线3："好奇→恍然"（权威解读型）
  [好奇（大师的矛盾行为）] → [认同（大师逻辑自洽）] → [升级（新认知框架）] → [亲近（大师也会犯错）] → [清醒（原则收束）]
  ```
  关键区别：曲线3不制造"不安"，全程由"好奇"驱动，安全感来自"我理解了大师的逻辑"
- **说服力来源**：实战经验感 > 数据权威感 > 逻辑严密感 > 人格信任感
- **认知节奏**：中等密度，每个数据点后都有大白话翻译。概念引入后给消化空间
- **信任建立方式**：展示自己的亏损经历 > 给出具体可执行的方案 > 精确引用权威材料 > 不卖课程 > 独立思考态度
- **读者的"安全感"**：行动清晰——"我知道下一步该做什么了"或认知升级——"我理解了大师的逻辑"

## 案例/数据生命点

- **核心案例的画面感**：亏损时的身体记忆（翻来覆去睡不着）、某次抄底的得与失
- **数据讲述方式**：用历史统计构建概率感（"标普500过去70年，年化收益约10%""83%的年份是正收益"），让数字变成信心
- **最具冲击力的对比**：反事实对比（"总投入少了18万、只辛苦了10年的策略A，60岁时的账户价值依然紧紧咬住了辛苦了25年、投入30万的策略B"）——让读者"算出来"差距
- **案例与结论的连接方式**：归纳总结（多个历史案例指向同一规律）+ 演绎推理（从框架推到行动）

## 市场语境适配

- **写作时须判断当前市场情绪**：恐慌 / 观望 / 震荡 / 狂热 / 冰点
- **文章与市场情绪的关系**：
  - 恐慌时：逆风提醒（打气 + 理性分析）
  - 狂热时：逆风提醒（泼冷水）
  - 震荡迷茫时：抽离观察（框架定心）
  - 冰点时：顺势共鸣（"现在就是最好的时候"）
- **时效性处理**：抄底/择时类内容须标注写作时点。理念类（定投、资产配置）长期有效

## 写作技术痕迹

- **标志性句式**：
  - "波动不等于风险，本金的永久性损失才是真正的风险"（否定+重定义型）
  - "先上车，再换座"（行动号召型）
  - "起初微若尘埃，后期势如破竹"（时间框架型）
  - "定投的本质，是把一生的现金流碎片，通过时间整合起来，源源不断地向复利系统注入燃料"（定义型）
  - "标普500赚得少一点，但亏起来也少一点"（对比型）
- **过渡手法**：从数据跳到观点时，用"换句话说"或比喻桥接；从观点跳到行动时，用"所以你应该"直接连接
- **金句分布规律**：开篇1句锚定 + 每个论证转折处1句 + 结尾1句收束
- **金句特征**：
  - 来源类型：实战血泪教训（60%）> 市场规律提炼 > 反直觉洞察 > 引用权威（40%）
  - 结构特点：绝大多数是**对称结构**或**否定+重定义结构**（"不是X，而是Y"）
  - 长度：1-2句话，不超过30个字为最佳

## 风险表达方式

三种层次，从基础到高级：

1. **标准化免责声明**（所有文章都有）："以上内容仅供学习参考，不构成任何投资建议"
2. **具体风险点提示**（融入操作步骤中）："汇丰对余额低于1万港币的账户收每月100港币管理费""长期国债ETF历史最大回撤曾超过-48%"
3. **以自身亏损作为风险教育载体**（最高级）："亏损2000万后，始明白……"
4. **以大师的失误作为风险教育载体**（权威解读型）："连巴菲特都会看走眼IBM""错过谷歌是他能力圈内的失误"——用大师的失败降低读者的盲目崇拜

**共同模式**：
- 风险提示不是附属品，而是嵌入在论证逻辑中——不单独列一段
- 语气统一为"善意的提醒"而非"恐吓"
- 免责声明不可省略，每篇至少一处"不构成投资建议"

## 给AI的创作方向

- **语气指引**：
  - 实战型：像一个自己亏过钱、但终于想通了的老朋友在饭桌上跟你讲投资——有教训、有方法、不卖弄、不教你"赚钱"，教你"少亏"。口语化但不随意
  - 权威解读型：像一个读过所有股东信的朋友在帮你"翻译"——有深度但不掉书袋。口语化、追问式、每段引用后跟大白话翻译
- **结构指引**：
  - 教程型：开场1-3句话进入主题 → 分步操作 → 个人方案展示 → 风险提示融入操作中 → 收束不超过3句话
  - 框架型：开场必须有钩子 → 框架建立 → 数据/案例填充 → 执行方案 → 哲学性升华结尾
  - 段落长度：每段不超过3-5句话（手机端阅读友好）
  - 金句密度：每800-1000字至少1个金句，放在段落结尾或章节收束处
- **人设指引**：
  - 实战型：必须展示个人痕迹（具体持仓比例、操作经历、亏损经历，至少择一）。避免"全能型专家"人设，最有效的是"走在你前面的同路人"
  - 权威解读型：不需要个人实战痕迹，但必须精确引用权威材料（标注年份、原文）。必须包含"大师也会犯错"段落，避免造神。最有效的人设是"帮你看懂大师的翻译者"
- **专业度校准**：
  - 必须引用的具体数据类型：历史收益率、最大回撤数据、费率对比数字。空谈"长期回报好"不如给"年化10.7%"
  - 允许使用但需解释的术语：EPS、ROE、市盈率、夏普比率。首次出现时一句话解释
  - 数据引用原则：给出具体来源（"根据先锋领航的数据"），不给来源的数据不如不给
- **特色模块：AI时刻**（Luna特色，可扩展到其他作者）：
  - 给出可直接复制的prompt模板，每个prompt解决一个具体认知盲区
  - 定位为"AI帮你做XX"，不是"AI替你决策"
- **需要避免的雷区**：
  - 使用"稳赚""保本"等违规用语或暗示保本保收益
  - 引用未经证实的小道消息或无法验证的个人收益截图
  - 制造FOMO（"再不买就晚了"）——取而代之的是"用定投化解择时难题"
  - 推荐具体个股或基金代码——所有文章只推荐指数基金/ETF
  - 不预测具体点位、不给出具体目标价
  - 用后视镜视角分析而不标注事后性质
